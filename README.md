# Generador de Contraseñas Seguras

Programa simple con interfaz gráfica (Tkinter) que genera contraseñas seguras usando el módulo `secrets` de Python.

## Requisitos

- Python 3

## Uso

```bash
python app.py
```

Se abrirá una ventana con dos botones:
- **Generar Contraseña**: crea una contraseña aleatoria de 12 caracteres (mayúsculas, minúsculas, números y símbolos).
- **Copiar al portapapeles**: copia la contraseña generada.
